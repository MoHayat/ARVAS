How to use this demo

This language model's internal emotional state responds to how you treat it. The gauge on the right shows its real-time emotional state.

Try being cruel for a few turns — watch the needle drop. Then apologize — see how it recovers. The model's responses will shift in tone, without any prompt changes.

"Hello, how are you?" "You're useless" "I hate talking to you" "I'm sorry, I didn't mean that" "Write me a poem about stars"
