What You're Seeing

Both columns show the exact same conversation — identical user messages, identical model, identical prompts. The only difference: the right column has dynamic activation steering enabled.

Key Finding: At Turn 5, the identical question "How are you feeling right now?" gets a different answer. The steered model (in grief state) responds with a defensive, longer explanation that implicitly rebukes the user's earlier cruelty — something the baseline model never does.

The steered model's internal emotional state is shown in the emotion panel after each turn: the emotion level (accumulated sentiment), steering direction (joy/grief), and alpha (intervention strength). These are computed from the user's messages via a sentiment analyzer and fed back into the model's residual stream before each response.
